WordPress.org plugin directory assets for CF7 Guests Field.

Use these files in the plugin SVN repository's top-level `assets/` directory:

- `icon-128x128.png`
- `icon-256x256.png`
- `banner-772x250.png`
- `banner-1544x500.png`

`icon-source.png` is the full-size generated source used to export the icon sizes.
`banner-source.png` is the full-size generated source used to export the banner sizes.
