WordPress.org plugin directory assets for Smart Guest Selector for Contact Form 7.

Use these files as the WordPress.org directory assets for the plugin:

- `icon-128x128.png`
- `icon-256x256.png`
- `banner-772x250.png`
- `banner-1544x500.png`

`icon-source.png` is the full-size generated source used to export the icon sizes.
`banner-source.png` is the full-size generated source used to export the banner sizes.
